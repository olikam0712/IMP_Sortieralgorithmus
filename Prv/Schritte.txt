Bubble Sort:
    1. Starte mit einer ungeordneten Liste von Elementen.
    2. Vergleiche das erste Element mit dem zweiten Element.
    3. Wenn das erste Element größer ist als das zweite Element, tausche sie.
    4. Vergleiche das zweite Element mit dem dritten Element und tausche sie gegebenenfalls.
    5. Fahre mit diesem Vergleichs- und Tauschprozess fort, bis das größte Element an das Ende der Liste "aufsteigt".
    6. Wiederhole die Schritte 2-5 für alle verbleibenden Elemente in der Liste.
    7. Nach einem vollständigen Durchlauf ist das größte Element am Ende der Liste.
    8. Wiederhole die Schritte 2-7, bis die gesamte Liste sortiert ist.
Insertion Sort:
    1. Starte mit einer ungeordneten Liste von Elementen.
    2. Betrachte das zweite Element in der Liste.
    3. Vergleiche das zweite Element mit dem vorherigen Element.
    4. Wenn das vorherige Element größer ist, verschiebe es eine Position nach rechts.
    5. Wiederhole den Vergleich und das Verschieben mit jedem vorherigen Element, bis das richtige Einfügeposition gefunden ist.
    6. Füge das aktuelle Element an der gefundenen Position ein.
    7. Wiederhole die Schritte 2-6 für alle verbleibenden Elemente in der Liste.
    8. Nachdem alle Elemente eingefügt wurden, ist die Liste sortiert.
Selection Sort:
    1. Starte mit einer ungeordneten Liste von Elementen.
    2. Finde das kleinste Element in der Liste.
    3. Tausche das kleinste Element mit dem ersten Element in der Liste.
    4. Betrachte den Rest der Liste (ohne das erste Element) als eine neue Teilmenge.
    5. Finde das kleinste Element in dieser Teilmenge.
    6. Tausche das kleinste Element mit dem zweiten Element in der Liste.
    7. Wiederhole die Schritte 4-6, bis die gesamte Liste sortiert ist.
    8. Nach Abschluss des Sortierprozesses ist die Liste in aufsteigender Reihenfolge geordnet.
